API Reference

RESTful API documentation for developers integrating with DreamBiz.

Authentication

API Key Authentication

Include your API key in the request header:

Authorization: Bearer YOUR_API_KEY

Obtaining API Key

  1. Navigate to Settings â†’ API Access
  2. Click "Generate API Key"
  3. Copy and store securely
  4. Never share or commit to version control

Base URL

https://your-domain.com/api/v1/

Endpoints

Invoices

GET /api/v1/invoices/ - List invoices
POST /api/v1/invoices/ - Create invoice
GET /api/v1/invoices/{id}/ - Get invoice
PUT /api/v1/invoices/{id}/ - Update invoice
DELETE /api/v1/invoices/{id}/ - Delete invoice

Expenses

GET /api/v1/expenses/ - List expenses
POST /api/v1/expenses/ - Create expense
GET /api/v1/expenses/{id}/ - Get expense

Customers

GET /api/v1/customers/ - List customers
POST /api/v1/customers/ - Create customer

Response Format

{
  "success": true,
  "data": {...},
  "message": "Success",
  "timestamp": "2025-11-11T10:30:00Z"
}

Error Codes

  • 200 - Success
  • 201 - Created
  • 400 - Bad Request
  • 401 - Unauthorized
  • 403 - Forbidden
  • 404 - Not Found
  • 500 - Server Error

Rate Limits

  • 1000 requests per hour per API key
  • 100 requests per minute
  • Bulk endpoints have lower limits

Important: Keep your API key secure. Rate limits apply to prevent abuse.
