Troubleshooting

Common issues and solutions to help you resolve problems quickly.

Login Issues

❌ Cannot log in

Solutions:

  1. Reset password via "Forgot Password" link
  2. Clear browser cache and cookies
  3. Try a different browser
  4. Contact admin to verify account status

❌ Two-factor authentication not working

Solutions:

  1. Verify device time is correct
  2. Use backup codes
  3. Contact support to reset 2FA

Invoice Issues

❌ Invoice not sending via email

Solutions:

  1. Verify email address is correct
  2. Check spam/junk folder
  3. Verify SMTP settings
  4. Try manual download and send

❌ Wrong tax calculation

Solutions:

  1. Verify tax rate settings
  2. Check customer tax exemption status
  3. Review product tax categories

Bank Reconciliation Issues

❌ Transactions not matching

Solutions:

  1. Check date ranges match
  2. Verify amounts match exactly
  3. Look for timing differences
  4. Check for duplicate entries

AI Insights Issues

❌ "OpenAI API key not configured"

Solutions:

  1. Add API key to .env file
  2. Verify key is correct (starts with sk-)
  3. Check OpenAI account has credits
  4. Restart server after adding key

❌ Insights not generating

Solutions:

  1. Verify you have financial data in system
  2. Check OpenAI service status
  3. Run: python manage.py test_openai
  4. Check browser console for errors

Performance Issues

❌ Slow loading times

Solutions:

  1. Clear browser cache
  2. Check internet connection speed
  3. Reduce date range in reports
  4. Contact support if persistent
