Bank Reconciliation

Match accounting records with bank statements to ensure accuracy and identify discrepancies.

Reconciliation Process

1. Import Statement
2. Match Transactions
3. Resolve Differences
4. Complete

Importing Bank Statements

Methods Available

  • Manual Entry: Type in transaction details
  • CSV Import: Upload bank statement file
  • API Integration: Direct bank connection (coming soon)

Matching Transactions

Automatic Matching

System automatically matches transactions by:

  • Exact amount and date matching
  • Reference number matching
  • Fuzzy matching for descriptions
  • Batch matching capabilities

Manual Matching

  1. Review unmatched transactions
  2. Select corresponding entries
  3. Confirm match
  4. Add notes if needed

Common Discrepancies

Timing Differences

Checks not cleared, deposits in transit

Recording Errors

Wrong amounts, duplicate entries, missing transactions

Bank Fees

Service charges, interest, transaction fees

Best Practices

  • Reconcile monthly for accuracy
  • Review all unmatched items
  • Investigate large differences immediately
  • Document all adjustments
  • Maintain reconciliation reports for audits
